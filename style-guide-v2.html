07 · Logo System

12 variants. One identity.

All logos are black vector marks — color is applied via context (cream or dark surface). Never stretch, recolor, or add effects. Use the lightest/simplest variant that fits the space.

Color rule: Logos are always monochrome. On cream or light surfaces — black as-is. On dark/soil surfaces — white (achieved via CSS filter: invert(1) or by using the white-fill version). Never use Terracotta or Olive Haze on the logomark itself.
